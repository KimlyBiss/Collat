Это программа для вычисления последовательности Коллатца, которая направлена на вычисление числовой последовательности, начиная с введенного числа, применяя следующее правило: если число четное, оно делится на 2, если число нечетное, умножается на 3 и добавляется 1. Функция возвращает список чисел этой последовательности.
